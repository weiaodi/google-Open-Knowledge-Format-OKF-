# Knowledge Base Index

## builtin (1 concepts)

- [Promise](./builtin/promise.md) — 代表异步操作最终完成或失败的对象，是 async/await 的底层机制

## patterns (1 concepts)

- [闭包 (Closure)](./patterns/closure.md) — 闭包是函数与其词法作用域的绑定，允许函数记住并访问外部函数的变量，即使外部函数已经执行完毕。

## references (1 concepts)

- [async function](./references/async_function.md) — 声明一个异步函数，该函数隐式返回 Promise 并允许在函数体中使用 await 关键字。

## syntax (1 concepts)

- [Arrow Functions](./syntax/arrow_function.md) — ES6 引入的简洁函数语法，具备词法 this 绑定，不能用作构造函数。

## topics (1 concepts)

- [Asynchronous JavaScript](./topics/async.md) — 事件循环、回调、Promise、async/await 构成的 JS 异步编程模型
