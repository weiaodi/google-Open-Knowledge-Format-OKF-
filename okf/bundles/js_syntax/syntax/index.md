# syntax

* [Arrow Functions](arrow_function.md) - 箭头函数是 ES2015 引入的简洁函数语法，提供词法 this 绑定，不可用作构造函数。
* [async / await](async_await.md) - 基于 Promise 的语法糖，让异步代码看起来像同步代码，避免回调地狱
* [Class Syntax](class.md) - 基于原型的 class 语法糖，支持 extends、super、static 成员和私有字段 (#)。
* [默认参数 (Default Parameters)](default_params.md) - 在函数定义时为参数指定默认值，当实参为 undefined 或未传入时自动生效。
* [Destructuring Assignment](destructuring.md) - 从数组或对象中提取值，并直接赋给变量的简洁语法，是 ES2015 (ES6) 引入的重要特性。
* [let & const](let_const.md) - ES2015 引入的块级作用域变量声明，let 用于可重新赋值的变量，const 用于不可重新绑定的常量绑定。
* [Spread & Rest Operator (...)](spread_rest.md) - 展开运算符(...)用于展开可迭代对象（数组、对象等），剩余参数语法用于收集函数参数中多余的参数为数组。
* [Template Literals](template_literals.md) - 反引号字符串，支持嵌入表达式插值、多行文本和标签模板函数。
* [var declaration](var.md) - 函数作用域变量声明，存在变量提升（hoisting），是 ES1 以来的传统声明方式。
